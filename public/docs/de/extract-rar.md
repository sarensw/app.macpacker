# RAR-Dateien auf macOS entpacken

RAR ist eines der beliebtesten Archivformate, wird aber von macOS nicht nativ unterstützt. MacPacker macht es einfach, RAR-Dateien auf Ihrem Mac zu entpacken.

## Voraussetzungen

- macOS 13.5 oder neuer
- MacPacker installiert über [Homebrew](https://brew.sh) oder als direkter Download

## Schritt-für-Schritt-Anleitung

### 1. MacPacker installieren

Der einfachste Weg, MacPacker zu installieren, ist über Homebrew:

```bash
brew install --cask macpacker
```

### 2. RAR-Datei öffnen

Doppelklicken Sie einfach auf eine `.rar`-Datei, und MacPacker öffnet sie automatisch.

### 3. Inhalte entpacken

Sobald das Archiv geöffnet ist, können Sie:

- **Alle Dateien entpacken** mit dem Entpacken-Button
- **Einzelne Dateien vorab ansehen**
- **Verschachtelte Archive navigieren**
