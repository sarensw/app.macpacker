# MacPacker - Archivmanager für macOS

Zeigen Sie Dateien aus beliebigen Archivformaten auf macOS an und entpacken Sie diese. Kostenlos, Open Source und auf Einfachheit ausgelegt.

## Hauptfunktionen

**Vorschau verschachtelter Archive**
Sehen Sie in Archive hinein, ohne sie zu entpacken – auch wenn sie in anderen Archiven verschachtelt sind. Navigieren Sie durch ZIP-Dateien, die RAR-Dateien enthalten, die wiederum 7z-Dateien enthalten, ohne etwas auspacken zu müssen.

**Einzelne Dateien extrahieren**
Extrahieren Sie nur die Dateien, die Sie benötigen, anstatt alles zu entpacken. Sparen Sie Zeit und Speicherplatz.

**Über 25 unterstützte Formate**
Funktioniert mit allen gängigen Archiv- und Disk-Image-Formaten:
- **Kompression:** ZIP, RAR, 7z, GZIP, LZ4, LZX, SEA, XAR
- **Disk-Images:** DMG, ISO, VDI, VMDK, VHDX, QCOW2, APFS, FAT, NTFS, VHD
- **Archive:** TAR, TAR.BZ2, TAR.LZ4, TAR.XZ, TAR.Z, CPIO, SquashFS
- **Andere:** ARJ, CAB, CHM, LHA, LZH, SIT

**Quick Look-Integration**
Zeigen Sie Archive direkt im Finder mit Quick Look (Leertaste) an.

**Mehrsprachige Unterstützung**
Verfügbar in Englisch, Deutsch, Französisch, Italienisch, Russisch, Ukrainisch, Chinesisch (vereinfacht) und Persisch (Farsi). Community-Übersetzungen willkommen.

## Systemanforderungen

- macOS 13.5 oder neuer
- 5 MB Speicherplatz

## Was ist neu

Version 0.13 enthält verbesserte Formatunterstützung, Quick Look-Integration und erweiterte Navigation verschachtelter Archive.

## Kostenlos & Open Source

MacPacker ist vollständig kostenlos ohne Werbung, Tracking oder Abonnements. Der Quellcode ist auf GitHub unter einer Open-Source-Lizenz verfügbar. Von der Community für die Community entwickelt.

## Datenschutz

MacPacker sammelt keine Daten. Keine Analysen, kein Tracking, keine Telemetrie. Was auf Ihrem Mac passiert, bleibt auf Ihrem Mac.
