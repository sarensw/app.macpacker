# MacPacker - macOS 归档管理器

在 macOS 上预览和解压任何归档格式的文件。免费、开源且简单易用。

## 主要功能

**嵌套归档预览**
无需解压即可查看归档内容——即使它们嵌套在其他归档中。浏览包含 RAR 文件的 ZIP 文件，而 RAR 文件又包含 7z 文件，完全无需解压任何内容。

**提取单个文件**
仅提取您需要的文件，而不是解压所有内容。节省时间和磁盘空间。

**支持 25+ 种格式**
适用于所有常见的归档和磁盘映像格式：
- **压缩格式：** ZIP、RAR、7z、GZIP、LZ4、LZX、SEA、XAR
- **磁盘映像：** DMG、ISO、VDI、VMDK、VHDX、QCOW2、APFS、FAT、NTFS、VHD
- **归档格式：** TAR、TAR.BZ2、TAR.LZ4、TAR.XZ、TAR.Z、CPIO、SquashFS
- **其他格式：** ARJ、CAB、CHM、LHA、LZH、SIT

**快速查看集成**
在访达中直接使用快速查看（空格键）预览归档文件。

**多语言支持**
提供英语、德语、法语、意大利语、俄语、乌克兰语、简体中文和波斯语（法尔西语）版本。欢迎社区翻译。

## 系统要求

- macOS 13.5 或更高版本
- 5 MB 磁盘空间

## 免费且开源

MacPacker 完全免费，无广告、无跟踪、无订阅。源代码在 GitHub 上以开源许可证提供。由社区构建，为社区服务。

## 隐私

MacPacker 不收集任何数据。无分析、无跟踪、无遥测。您 Mac 上发生的事情仅留在您的 Mac 上。
