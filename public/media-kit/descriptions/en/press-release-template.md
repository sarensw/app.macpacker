## About MacPacker

MacPacker is a free, open-source archive manager for macOS that enables users to preview and extract files from 25+ archive formats without extracting nested archives. Built with a focus on simplicity and user privacy, MacPacker supports common compression formats (ZIP, RAR, 7z), disk images (DMG, ISO, VMDK, QCOW2), and legacy archive types.

Key capabilities include:
- Nested archive preview without extraction
- Single-file extraction from any archive
- Quick Look integration in macOS Finder
- Support for 25+ archive and disk image formats
- Multilingual interface (8 languages)
- Zero tracking or telemetry

MacPacker is available as a free download via Homebrew, direct download, or the Mac App Store. The application requires macOS 13.5 or newer and occupies just 5 MB of disk space.
